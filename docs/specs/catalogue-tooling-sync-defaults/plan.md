# Plan: Catalogue Tooling — Sync-Defaults

- **Spec:** [`spec.md`](spec.md)
- **Status:** Drafting

## Approach

Four tasks, sequential: (1) minimal TOML emitter + `compile_defaults` pure function;
(2) `check_defaults` and `write_defaults` filesystem operations; (3) CLI wiring;
(4) `catalogue verify` integration. Each task is independently testable. No
external dependencies.

## Constraints

- ini-005 brief Bucket 4; Python 3.11 stdlib only.
- Reuse `config.py` validators for Artifactory and preferred-adapter; no duplicates.
- `install-defaults.toml` is the only output format.
- Must not break any existing test.

## Construction tests

- `test_compile_defaults_public` — Artifactory disabled; assert header, `preferred_adapter`, source.
- `test_compile_defaults_enterprise` — Artifactory enabled; assert `base-url`, `repository`, `bundle`, `channel`.
- `test_compile_defaults_deterministic` — call twice; assert identical output.
- `test_compile_defaults_no_credentials` — assert no `token`, `password`, `key=`, `secret` in output.
- `test_check_clean` / `test_check_drift` / `test_check_absent` — `ok=True`, `ok=False`, `ok=False`.
- `test_write_atomic` — assert `.tmp` gone; file content correct.
- `test_write_path_jail` / `test_write_symlink_guard` — assert `CatalogueConfigError`.
- `test_cli_check_clean` / `test_cli_check_drift` — subprocess; exit 0 / exit 1.
- `test_verify_surfaces_drift` — `verify_catalogue` with drifted file; assert ERROR diagnostic.

## Design (LLD)

### Module: `catalogue_tooling/defaults.py`

```python
def compile_defaults(config: CatalogueConfig) -> str:
    """Pure function. Deterministic TOML string. No FS I/O."""

def check_defaults(root: Path) -> SyncDefaultsResult:
    """Read-only. ok=True iff current file byte-matches compile_defaults output."""

def write_defaults(root: Path) -> SyncDefaultsResult:
    """Atomic write. Enforces path jail + symlink guard before writing."""
```

### Generated TOML structure

```toml
# Generated by agentbundle catalogue sync-defaults — source of truth: catalogue.toml
[organization]
preferred_adapter = "<distribution.agentbundle.preferred-adapter>"

[organization.artifactory]
enabled = false

[defaults]
source = "<distribution.agentbundle.default-source>"
```

When `artifactory.enabled = true`, `[organization.artifactory]` expands to include
`base-url`, `repository`, `bundle`, `channel`. The `source` key under `[defaults]`
is omitted when `default-source` is absent from config.

### TOML emitter

Grep for an existing minimal emitter (`_emit_toml` or similar); reuse if found.
Otherwise add `_emit_toml(sections: dict[str, dict]) -> str` as a private function
in `defaults.py`. Contract: `str` and `bool` scalars only; sections in insertion
order; no arrays or nested tables beyond `[organization.artifactory]`.

### Atomic write

```python
tmp = output_path.with_suffix(".tmp")
tmp.write_text(content, encoding="utf-8")
tmp.rename(output_path)
```

Path jail: `output_path.resolve().is_relative_to(root.resolve())`.
Symlink guard: `not output_path.is_symlink()`. Both raise `CatalogueConfigError`.

### `SyncDefaultsResult`

Inherits `CommandResult`; adds no new fields. `operation` = `"check"` or `"write"`.
Drift and path-jail findings surface as `diagnostics` entries.

## Tasks

### T1: TOML emitter + `compile_defaults`

**Verification mode:** TDD

**Tests:** `test_compile_defaults_public`, `test_compile_defaults_enterprise`,
`test_compile_defaults_deterministic`, `test_compile_defaults_no_credentials`

**Approach:** Grep for an existing emitter; reuse or add `_emit_toml` to `defaults.py`.
Implement `compile_defaults`: map `CatalogueConfig` fields to three sections, call
emitter, prepend header comment. Tests use inline `CatalogueConfig` instances, no
filesystem.

**Depends on:** Foundation spec (CatalogueConfig, SyncDefaultsResult available)

---

### T2: `check_defaults` and `write_defaults`

**Verification mode:** TDD

**Tests:** `test_check_clean`, `test_check_drift`, `test_check_absent`,
`test_write_atomic`, `test_write_path_jail`, `test_write_symlink_guard`

**Approach:** `check_defaults` calls `load_catalogue_config`, `compile_defaults`,
reads existing file, byte-compares. `write_defaults` enforces path jail and
symlink guard, then writes atomically. Tests use `tmp_path`; symlink test uses
`Path.symlink_to`.

**Depends on:** T1

---

### T3: CLI wiring

**Verification mode:** Goal-based check

**Tests:** `test_cli_check_clean` (exit 0), `test_cli_check_drift` (exit 1),
`test_cli_write` (exit 0, file updated)

**Approach:** Replace the `sync-defaults` stub by adding
`agentbundle/commands/catalogue_sync_defaults.py`. Handler dispatches to
`check_defaults` or `write_defaults` per `--check`/`--write` flag, serialises
`SyncDefaultsResult` to JSON on stdout, exits `0 if result.ok else 1`.

**Depends on:** T2
