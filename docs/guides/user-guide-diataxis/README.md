# `user-guide-diataxis` — guides

`user-guide-diataxis` is the docs skeleton for your product: the four [Diátaxis](https://diataxis.fr/) kinds — tutorials, how-to, reference, explanation — plus the `new-guide` skill that lands every new page on the right side of the line. One kind per page, never blended. The link-out discipline does the rest.

New here? Read [About the Diátaxis framework](explanation/the-diataxis-framework.md) first — it's the model. Then write your first page with [How to write a guide](how-to/write-a-guide.md).
